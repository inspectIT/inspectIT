Documentation

You know, we also do not like to read product documentation in order to get familiar with some tool. This is why inspectIT tries to provide very intuitive approach for performance monitoring. But, we still think that an up-to-date documentation can help to understand all the possibilities that inspectIT offers.

We invest quite some time in keeping the documentation up-to-date and try to explain all the features in detail. Still, if you find something is missing or hard to understand you can write us an e-mail and we will help you.

Documentation Content

In the inspectIT Documentation you can find a brief Overview of the inspectIT approach. The Installation Guide describes how to integrate inspectIT with different technologies. In-depth information about configuration possibilities is available in the Configuration Guide, while the Administration Guide can help in maintaining the inspectIT components.

If you want to play around with inspectIT, the Samples section provides sample applications with step-by-step guides. If you have any additional questions you can also check the FAQ collection.
